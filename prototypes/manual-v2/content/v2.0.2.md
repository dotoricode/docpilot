# DocPilot v2.0.2

> 공개일: 2026-07-18 · 설치 파일과 태그는 GitHub Release `v2.0.2`에서 제공합니다.

DocPilot 2.0.2는 앱 종료, 워크스페이스 보안 경계, 동시 저장과 외부 변경 처리에 집중한 안정성 릴리스입니다. Intel Mac과 Apple Silicon용 패키지를 각각 제공합니다.

## 종료와 리소스 정리

- 앱 종료 시 Bridge, watcher, AsciiDoc worker, socket과 SSE 연결을 정리합니다.
- 터미널과 Agent 자식 프로세스를 순서대로 종료하고 종료 중 새 리소스를 받지 않습니다.
- 반복 종료와 정리 지연 상황에서도 앱이 남아 있지 않도록 수명주기를 보강했습니다.

## 워크스페이스 보안

- Bridge capability token, Host와 Origin 정책을 fail-closed 방식으로 적용합니다.
- traversal과 symlink escape, malformed JSON, 허용되지 않은 요청을 차단합니다.
- 저장 commit 직전에 대상 파일을 다시 확인해 symlink 교체가 외부 파일에 닿지 않도록 합니다.

## 저장과 충돌 보호

- 동일 파일의 다른 split view에 미저장 초안이 있으면 peer 저장 결과가 초안을 덮어쓰지 않습니다.
- 외부 변경 충돌에서는 사용자 확인 뒤에만 로컬 초안을 디스크에 반영합니다.
- cross-volume trash publish가 실패하면 원본을 보존하고 임시 staging 파일을 정리합니다.

## macOS 패키지

- Apple Silicon용 `DocPilot-2.0.2-arm64.dmg`를 제공합니다.
- Intel Mac용 `DocPilot-2.0.2-x64.dmg`를 제공합니다.
- 앱 실행 파일과 native PTY 모듈이 각 DMG의 아키텍처와 일치하는지 검사합니다.
- 배포 파일은 ad-hoc 서명되며 Apple 공증은 적용되지 않았습니다.

## 검증

- TypeScript, 42개 단위·통합 테스트, 종료 수명주기와 dependency audit
- 실제 macOS Electron AsciiDoc 보안 및 외부 변경 E2E
- x64/arm64 DMG, app executable, native PTY와 packaged bridge 검사
- Intel 패키지 실행 smoke와 아키텍처별 다운로드 계약
