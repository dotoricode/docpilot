# DocPilot v2.0.1

> 공개일: 2026-07-16 · 설치 파일과 태그는 GitHub Release `v2.0.1`에서 제공합니다.

DocPilot 2.0.1은 Markdown과 AsciiDoc 프리뷰의 표시 문제를 보정하고, 처음 실행하는 사용자가 테마·본문 폭·줄 번호·터미널을 더 쉽게 이해할 수 있도록 기본 동작을 정리한 수정 릴리스입니다.

## 문서 프리뷰

- Markdown과 AsciiDoc 제목의 크기와 굵기를 동일한 시각 단계로 맞췄습니다.
- 한국어 제목이 시스템 글꼴에 따라 달라지지 않도록 Noto Sans KR 가변 글꼴을 앱에 포함했습니다.
- AsciiDoc NOTE의 긴 권한명과 인라인 코드가 패널 밖으로 넘치지 않도록 줄바꿈을 보정했습니다.
- 여러 자리 줄 범위가 하이픈에서 줄바꿈되어 겹치지 않도록 한 줄로 표시합니다.

## 처음 실행할 때 달라지는 점

- 앱과 파일 열기 화면은 macOS 시스템 테마를 기본값으로 사용합니다.
- 프리뷰 본문은 사용 가능한 최대 폭보다 한 단계 좁게 시작합니다.
- 폭 조절선은 마우스를 올리지 않아도 은은하게 보입니다.
- 줄 번호는 기본적으로 꺼져 있고 상단 `Line numbers` 스위치에서 바로 켤 수 있습니다.

## 터미널과 워크스페이스

- 터미널을 닫으면 홈과 문서 화면 우측 하단에 다시 열기 버튼이 나타납니다.
- 워크스페이스 이름 옆의 동작하지 않는 화살표 표시를 제거했습니다.

## 매뉴얼과 설치

- PR #19의 실제 앱 기반 데모와 PR #22의 Vercel 배포 allowlist 수정이 포함됩니다.
- 매뉴얼의 Download 버튼은 최신 GitHub Release의 DMG 자산을 직접 엽니다.
- 현재 공개 DMG는 Intel Mac(x64)용이며 코드 서명·공증되지 않았습니다.

- Markdown·AsciiDoc 프리뷰 UI 회귀 검사
- 프리뷰 줄 번호, 폭 조절과 테마·복사 흐름 검사
- Intel x64 패키지 실행 파일과 네이티브 PTY 아키텍처 검사
- 매뉴얼 콘텐츠·경로·미디어 계약과 production build 검사
